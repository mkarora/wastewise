# WasteWise Frontend

Angular frontend application for WasteWise.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:4200`

## Build

To build for production:
```bash
npm run build
```

The built files will be in the `dist/wastewise-frontend` folder.
